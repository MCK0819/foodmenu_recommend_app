# foodmenu_recommend_app
flutter를 사용하여 메뉴 추천 어플

기본적으로 필요한 기능
1. 메뉴 추천 기능 : 사용자가 원하는 음식 종류, 가격 대, 거리 등을 입력하면 해당 조건에 맞는 식당 메뉴를 추천해주는 기능이 필요
2. 지도 검색 기능: 사용자가 위치를 입력하면, 해당 지역의 식당 정보를 지도에서 검색해주는 기능이 필요
3. 검색 기능: 사용자가 원하는 음식 종류나 가게 이름 등을 검색할 수 있는 검색 기능이 필요
4. 리뷰 기능: 사용자가 식당에 대한 리뷰를 작성할 수 있는 기능이 필요
5. 북마크 기능: 사용자가 자주 가는 식당을 저장하고 추천 기능에서 우선적으로 보여줄 수 있는 북마크 기능이 필요
6. 공유 기능: 사용자가 추천 받은 메뉴나 리뷰를 다른 사용자와 공유할 수 있는 기능이 필요
7. 로그인 기능: 사용자가 자신의 정보를 등록하고 로그인하여 개인화된 추천을 받을 수 있는 기능이 필요
